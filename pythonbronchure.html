Bootcamp Overview

Python for Data Analysis and Visualization is a structured, mentor-guided 10-week live cohort program designed to take learners from foundational Python concepts to professional-grade data analysis, visualization, and dashboard deployment.

  • Practical, hands-on learning
  • Real datasets and applied analytics
  • Strong data storytelling principles
  • Reproducible workflows using modern tools

Participants graduate with job-ready skills and a public analytics portfolio.

Bootcamp Features

  • Weekly live instructor-led sessions
  • Structured pre-work and guided code-alongs
  • Hands-on workshop assignments with feedback
  • Mentor-guided practical sessions
  • Private learning community
  • Capstone dashboard project
  • SI Analytics Certificate of Completion
  • Lifetime access to curated resources

Duration: 10 Weeks
Format: Live + Guided Self-Paced

How the Bootcamp Works

  • Pre-work: 2–3 hours of videos and code-alongs
  • Live Workshop: 2-hour instructor-led session
  • Post-work: Applied assignment with feedback
  • Support Session: Optional alternate live session

Course Modules Overview

  • Module 1: Python Foundations
  • Module 2: Data Visualization
  • Module 3: Data Wrangling with Pandas
  • Module 4: Git & GitHub
  • Module 5: Dashboards with Quarto
  • Module 6: Capstone Project
